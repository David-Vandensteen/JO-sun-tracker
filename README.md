# JO-sun-tracker

## Matériel pour un groupe de panneaux solaire
- 1 carte Arduino UNO (5v) ou 1 carte Arduino NANO (3,3v) ?
- 1 module L298N (peut piloter 2 moteurs 12v)
- 2 modules LDR LM393 (photocellule)
- 1 anémomètre (ref ?)
- 2 boutons poussoir
- 1 bouton switch
- 2 vérins 650mm 12v
